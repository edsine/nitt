API Documentation
This documentation provides details on the endpoints available in the API along with their descriptions, request methods, and any required parameters or authentication.

Authentication

Login
URL: /auth/login
Method: POST
Description: Endpoint for user authentication.
Parameters:
email (string, required): User's email address.
password (string, required): User's password.

Logout
URL: /auth/logout
Method: POST
Description: Endpoint for user logout. Requires authentication.
Authentication: Requires authentication token.

Register
URL: /auth/register
Method: POST
Description: Endpoint for user registration.
Parameters:
name (string, required): User's name.
email (string, required): User's email address.
password (string, required): User's password.
password_confirmation (string, required): Confirmation of user's password.

Recover Password
URL: /auth/recover
Method: POST
Description: Endpoint for recovering forgotten password.
Parameters:
email (string, required): User's email address.

Reset Password
URL: /auth/reset
Method: POST
Description: Endpoint for resetting password after recovery.
Parameters:
email (string, required): User's email address.
token (string, required): Token received for password reset.

Users

Get User
URL: /user
Method: GET
Description: Retrieves the authenticated user's information.
Authentication: Requires authentication token.

Update User Profile
URL: /users/update_profile/{id}
Method: PUT
Description: Updates the profile of the specified user.
Parameters:
id (integer, required): User ID.
Other user profile fields.
Update User Profile Image

URL: /users/update_profile_image/{id}
Method: POST
Description: Updates the profile image of the specified user.
Parameters:
id (integer, required): User ID.
image (file, required): Image file for profile picture.
Change Password

URL: /users/change_password/{id}
Method: POST
Description: Changes the password of the specified user.
Parameters:
id (integer, required): User ID.
password (string, required): New password for the user.
